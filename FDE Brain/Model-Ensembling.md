# Model Ensembling

## Summary

Combining the outputs (predictions) of multiple independent models for a single query, rather than combining their internal parameters.

## Core Idea

Ensembling improves predictive performance by leveraging the diverse strengths of multiple models. The final answer is generated by aggregating the outputs (e.g., majority vote or a trainable ML module).

## Practical Use

Use ensembling when maximum accuracy is paramount and inference cost is secondary. For example, using three different LLMs to generate three answers and then using a simple majority vote to select the final response.
