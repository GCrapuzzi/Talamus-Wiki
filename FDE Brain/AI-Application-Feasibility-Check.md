# AI Application Feasibility Check

## Summary

A set of initial questions required before committing resources to building an AI application, ensuring the problem is solvable and necessary.

## Core Idea

Before building, an engineer must answer fundamental questions: Is the application necessary? Is AI needed? Do I have to build this application myself? This prevents scope creep and wasted effort.

## Practical Use

Apply this checklist during the initial discovery phase of any project to validate the business need and technical feasibility of using an LLM.
