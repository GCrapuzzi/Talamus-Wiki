# Crawl-Walk-Run Automation Model

## Summary

A staged framework for increasing AI automation in a product, defining clear milestones for human involvement and system autonomy.

## Core Idea

Automation should be implemented incrementally to manage risk and build user trust. The transition moves from mandatory human involvement to direct, autonomous AI interaction.

## Practical Use

Use this model as a product roadmap: 1. Crawl (Human mandatory involvement). 2. Walk (AI interacts with internal employees). 3. Run (Increased automation, direct AI interaction with external users).
