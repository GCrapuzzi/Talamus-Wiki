# AI Development Focus Shift (Traditional ML vs Foundation Models)

## Summary

The shift from traditional Machine Learning (ML) to Foundation Models (FMs) changes the critical focus areas in the AI development lifecycle, increasing the importance of application development and evaluation.

## Core Idea

As models become more powerful and general (Foundation Models), the bottleneck shifts from model training/prompting to how the model is evaluated, integrated, and presented to the user. This increases the overlap with full-stack development.

## Practical Use

Use this framework to scope a project: If using FMs, allocate significant resources to robust evaluation pipelines and interface design, rather than solely focusing on prompt engineering or model training. This guides the team toward a full-stack AI approach.
