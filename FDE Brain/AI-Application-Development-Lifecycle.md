# AI Application Development Lifecycle

## Summary

A structured, multi-stage process for building and refining AI applications using foundation models, emphasizing iterative evaluation.

## Core Idea

Successful AI application development requires moving beyond simple prompting and systematically optimizing three core pillars: the model itself, the context provided, and the instructions given, all while maintaining a rigorous evaluation pipeline.

## Practical Use

Use this framework to scope a project, starting with a feasibility check (Is AI needed? Is the app necessary?), followed by iterative development focusing on Prompt Engineering, Context Construction (RAG/Agentic), and continuous Evaluation.
