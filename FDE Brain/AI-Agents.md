# AI Agents

## Summary

AI systems designed to plan, execute multi-step tasks, and interact with external tools (APIs, search engines) to achieve complex goals autonomously.

## Core Idea

Agents move AI beyond simple prompt-response interactions. They possess the ability to reason, plan, and use external resources, making them capable of performing complex, real-world tasks (e.g., booking a trip, managing a workflow) that require multiple steps and tool access.

## Practical Use

Design an agent workflow that requires multiple external calls (e.g., calling a search API, then a calendar API, then a payment API) to complete a business process like 'onboarding a new client' or 'generating a quarterly report'.
