# Distillation

Finetuning a small model to imitate the behavior of a larger model using data generated by the large model. Distills the larger model's knowledge into the smaller one.

**Distillation development path** (per OpenAI best practices):
1. Start with a small dataset and the strongest model you can afford → train the best possible model
2. Use this finetuned model to generate more training data
3. Use the synthetic dataset to train a cheaper, smaller model

Example: Grammarly's finetuned Flan-T5 outperformed a GPT-3 variant on writing assistant tasks while being **60× smaller**, using only 82K (instruction, output) pairs.

Distillation is discussed further alongside other data synthesis techniques in the context of training data acquisition.
