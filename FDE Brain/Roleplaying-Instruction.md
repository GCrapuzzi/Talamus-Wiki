# Roleplaying Instruction

## Summary

A common instruction type where the model is prompted to assume a specific fictional character, persona, or role to guide its tone, vocabulary, and perspective.

## Core Idea

Roleplaying is highly effective for controlling the style and context of the output. It can be used either for user interaction (entertainment) or as a prompt engineering technique to improve output quality and consistency.

## Practical Use

When the output needs a specific voice (e.g., 'Act as a 19th-century botanist' or 'Respond as a skeptical financial analyst'), explicitly define the persona, background, and constraints of that role in the system prompt to maximize adherence.
