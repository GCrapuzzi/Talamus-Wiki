# AI Agent Definition and Components

## Summary

An agent is any system characterized by its environment and the set of actions it can perform, defined by its ability to perceive its environment and act upon it.

## Core Idea

An agent's capability is fundamentally defined by the intersection of its operational environment (the use case/domain) and its available toolset (the actions it can execute).

## Practical Use

When designing an agent, first define the scope (environment, e.g., 'internet scraping' or 'internal database') and then inventory the necessary tools (e.g., 'web search API', 'SQL executor', 'Python interpreter').
