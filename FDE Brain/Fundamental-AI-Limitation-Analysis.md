# Fundamental AI Limitation Analysis

## Summary

A decision framework for determining if a real-world problem is due to inherent, fundamental limitations of current AI technology or if it can be solved with improved models and techniques.

## Core Idea

If a problem persists even with state-of-the-art models, it suggests a fundamental limitation (e.g., common sense reasoning, causality). Addressing these requires novel architectural or methodological shifts, rather than just scaling up existing models.

## Practical Use

When scoping a new AI project, use this framework to classify the problem. If it's fundamental, the solution must be highly innovative; if it's model-dependent, focus on data quality, prompt engineering, or fine-tuning.
