# Language Model (LM)

## Summary

A system that encodes statistical information about one or more languages, predicting the likelihood of a word or token appearing in a given context.

## Core Idea

LMs model the statistical relationships between tokens in a sequence. They function as prediction engines, determining the most probable next token based on preceding context.

## Practical Use

Used for text generation, predictive text, and understanding the statistical structure of natural language. Engineers use LMs to build applications that require understanding or generating human language.
