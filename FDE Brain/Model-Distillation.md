# Model Distillation

## Summary

A machine learning technique where a smaller, faster model (the student) is trained to replicate the performance of a larger, more complex model (the teacher).

## Core Idea

It reduces the computational overhead and deployment latency of powerful models while maintaining comparable accuracy, often by using data generated by the original, high-performing model.

## Practical Use

Creating a lightweight, edge-device version of a large language model (LLM) for faster inference or deployment on resource-constrained hardware.
