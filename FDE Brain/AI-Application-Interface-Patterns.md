# AI Application Interface Patterns

## Summary

AI applications can be deployed through diverse interfaces, ranging from traditional web/desktop apps to highly integrated, natural language channels.

## Core Idea

The choice of interface dictates the user experience, the required integration complexity, and the method of user feedback collection. The trend is toward embedding AI into existing workflows.

## Practical Use

When designing an AI product, map the intended user interaction (e.g., chat, voice, API call) to select the appropriate deployment pattern. Prioritize API integration for enterprise products (e.g., VSCode, Shopify) to maximize reach.
