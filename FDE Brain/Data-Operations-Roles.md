# Data Operations Roles

## Summary

The specialized roles and functions dedicated to the systematic collection, cleaning, labeling, and validation of training data, recognizing data handling as a dedicated engineering discipline.

## Core Idea

Data handling has evolved from a side task to a core, specialized function within AI companies. Dedicated roles (Data Labelers, Data Quality Engineers) are necessary to manage the complexity and scale of modern AI datasets.

## Practical Use

When scaling an AI project, budget and staffing for these roles must be treated as core infrastructure investments, not optional overhead. Define clear workflows for data labelers and quality engineers.
