# AI Use Case Taxonomy

## Summary

A structured classification of common AI applications, grouping them by function (e.g., Coding, Writing, Data Organization) and providing examples for both consumer and enterprise contexts.

## Core Idea

AI applications are not monolithic; they fall into distinct functional categories. Using a standardized taxonomy helps engineers scope projects, identify necessary tools, and understand the breadth of potential solutions.

## Practical Use

When starting a new project, use this taxonomy to map the desired functionality (e.g., 'Summarization' or 'Data Extraction') to a known pattern. This helps determine if the required capability is a standard foundation model task or requires specialized fine-tuning.
