# Vector Search

## Summary

A computational method that uses vector embeddings (numerical representations of data) to calculate the similarity between query vectors and stored document vectors, enabling semantic retrieval.

## Core Idea

Vector search allows systems to find information based on meaning (semantics) rather than just keyword matching, making it the backbone for advanced search and recommendation systems.

## Practical Use

Use vector search as the core retrieval mechanism for RAG systems when semantic understanding is critical. It is computationally intensive but offers superior performance compared to traditional term-based methods.
