# Specialized AI Coding Patterns

## Summary

A taxonomy of high-value, specialized coding tasks that can be automated or significantly assisted by foundation models.

## Core Idea

Instead of treating AI as a general coding replacement, treat it as a specialized tool for discrete engineering functions. Identifying the right pattern is key to maximizing productivity gains.

## Practical Use

When encountering a development bottleneck, map the task to a specialized pattern: 
1. **Data Extraction:** Use agents for structured data parsing from PDFs/web pages. 
2. **UI Generation:** Use screenshot-to-code tools for rapid frontend prototyping. 
3. **Code Translation:** Use specialized models for migrating code between languages/frameworks.
