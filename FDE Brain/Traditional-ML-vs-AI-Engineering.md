# Traditional ML vs AI Engineering

## Summary

Defines the fundamental differences in methodology and focus between building applications on traditional machine learning models and building applications on foundation models.

## Core Idea

Traditional ML focuses on structured data, feature engineering, and model training (e.g., tabular data). AI Engineering (Foundation Models) focuses on prompt engineering, context construction, and parameter-efficient finetuning, often dealing with unstructured data.

## Practical Use

When starting a project, determine if the core problem requires structured feature extraction (Traditional ML) or advanced reasoning/contextual understanding (AI Engineering). A real-world system often requires knowledge of both.
