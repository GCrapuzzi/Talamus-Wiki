# AI Engineering Scope Definition

## Summary

AI Engineering is defined as the practice of integrating AI models into existing software stacks, treating the models as core components within a broader software development lifecycle.

## Core Idea

AI engineering is fundamentally software engineering principles applied to systems that incorporate AI models. It requires not just model training, but robust deployment, integration, and maintenance within a functional product.

## Practical Use

When scoping a project, recognize that the challenge is not just model accuracy, but building the surrounding infrastructure (APIs, frontends, data pipelines) to make the model usable in a real-world product.
