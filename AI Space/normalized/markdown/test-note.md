# The Forward Deployed Engineer Mindset

The FDE engineer sits between [[customer requirements]] and shipped software. They translate ambiguous business goals into concrete technical decisions.

#fde-engineering #craft

Three principles guide the work:

1. Ship something useful in week one
2. Build feedback loops with the customer before optimizing
3. Refactor only after the second similar problem
